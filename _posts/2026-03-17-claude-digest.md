## 💡 Deep Dive

### 1. Software 3.0 시대와 Claude Code의 역할

**핵심:** Andrej Karpathy가 정의한 Software 3.0 패러다임에서 LLM은 단순한 모델이 아닌 '에이전트'로 진화하고 있습니다. Claude Code는 이러한 LLM의 능력을 실제 업무에 연결하는 '하네스(harness)' 역할을 수행합니다.

**공통 의견:** 기술 커뮤니티에서는 Claude가 단순 챗봇을 넘어 실제 코딩 작업을 자동화하는 도구로 인식되고 있습니다. MCP(Model Context Protocol), 스킬, 슬래시 명령어 등의 구조는 Software 1.0의 명시적 프로그래밍 개념을 새로운 방식으로 재해석한 것입니다.

**실무 적용:**

- Claude Code를 통해 버그 수정, 코드 리뷰, 자동화 스크립트 작성을 직접 실행 가능
- 프롬프트를 '프로그램'으로 생각하고 체계적으로 설계하기
- 팀 협업 환경에서 Claude의 에이전트 기능을 워크플로우에 통합

### 2. Claude vs ChatGPT: 차별화된 강점

**핵심:** Claude는 ChatGPT와 달리 '긴 문서 이해 및 정리 능력'에서 우월하며, 이를 바탕으로 일정 관리, 데이터 분석, 글쓰기 등 구체적인 업무 영역에서 더 강력한 성능을 보입니다.

**공통 의견:** 사용자들이 처음엔 ChatGPT 중심으로 생각하지만, 실제 사용해보면 Claude의 문맥 이해도와 정확성이 특정 작업에서 더 우수하다고 평가합니다. 특히 한국 사용자들 사이에서 "한 달 써보니 생각이 바뀌었다"는 후기가 증가하는 추세입니다.

**실무 적용:**

- 장문의 보고서, 계약서, 기술 문서 분석에 Claude 우선 사용
- 엑셀 Add-in으로 설치하여 데이터 정리 및 오류 확인 자동화
- 부업이나 글쓰기 작업에서 Claude의 문맥 유지 능력 활용

### 3. Claude 요금제 및 사용량 최적화 전략

**핵심:** 2026년 기준 Claude는 Free, Pro, Max, Team 등 다층화된 요금제를 제공하며, 최근 '사용량 2배 이벤트'를 통해 사용자 확대를 추진 중입니다. 각 요금제별로 Claude Code, Cowork, Google Workspace 연동 등의 기능 차이가 있습니다.

**공통 의견:** 개인 사용자는 Pro 요금제(월 20달러)에서 가성비를 느끼고 있으며, 팀 단위 사용은 Team 요금제로 협업 기능을 활용하는 추세입니다. 사용량 2배 이벤트는 한국 시간 기준으로 적용되는 시점을 정확히 파악하는 것이 중요합니다.

**실무 적용:**

- 개인 개발자/프리랜서: Pro 요금제로 Claude Code 활용
- 팀 협업: Team 요금제로 Cowork 기능 및 Google Workspace 연동 활성화
- 사용량 2배 이벤트 기간에 대량의 문서 분석, 코드 생성 작업 집중 처리
